# unfold

Notes from a practice that unfolds - portable harness-engineering and agent-first development practices refined through real projects.

This repository hosts the source for the unfold site, published via GitHub Pages at https://gneeek.github.io/unfold (may take a minute to come live after initial setup).

unfold is deliberately in-progress. Pages are shipped as stubs and fleshed out in subsequent small releases. A separate future repository named `legible` is reserved for battle-tested patterns that have matured through use on multiple projects.

Initial content is landing during v1.3.5 of the originating project.
