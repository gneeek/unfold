# Three work loops

Draft scheduled for a later v1.3.x release.

This page will describe the three modes of work that should not be conflated: choosing problems to solve, choosing how to solve problems, and choosing how to deliver. Applied in recurring loops, with feedback from delivery flowing back into problem selection.

For the current source of this framework, see the originating project's wiki page on how we work.
