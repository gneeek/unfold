- **[Hygiene and retrospective: different mechanisms](/unfold/notes/2026-04-20-hygiene-and-retrospective/)** (2026-04-20). Hygiene checks run before an action and verify state; retrospectives run after and capture learning. When something goes wrong the useful question is whether the fix is verifying-shaped or learning-shaped — retros cannot fix a hygiene-shaped problem.
- **[Keeping `git status` load-bearing](/unfold/notes/2026-04-20-git-status-as-load-bearing-signal/)** (2026-04-20). `git status` is a signal with a signal-to-noise ratio. The fix is not more careful reading; it is `.gitignore` curation as recurring hygiene, so every untracked file either gets committed soon or disappears from the signal entirely.
- **[Deliberate session-end at workstream seams](/unfold/notes/2026-04-20-session-end-at-seams/)** (2026-04-20). When one agent session spans multiple workstreams, the seams are where mistakes happen. Deliberately ending the session at a seam turns an accidental boundary into an intentional checkpoint and forces the hygiene check on resume.
- **[The last responsible moment under a permanence rule](/unfold/notes/2026-04-20-last-responsible-moment-and-permanence/)** (2026-04-20). Poppendieck's last responsible moment is a judgement call unless the project has a permanence rule — a content-change rule, append-only log, or immutable tag — that defines the moment sharply and gives the review window structural weight.
- **[Ceremonies on the OODA loop: retro, planning, sprint](/unfold/notes/2026-04-20-ceremonies-on-the-ooda-loop/)** (2026-04-20). A sharper reading of the retro/planning/sprint mapping — orient bridges retro and planning, which is why retro-before-planning is structural rather than conventional, and which tells you when to run a retro.
- **[Parallel branches, parallel sessions](/unfold/notes/2026-04-11-parallel-branches-parallel-sessions/)** (2026-04-11). Two patterns for keeping parallel streams of work moving without blocking: git worktrees for branch isolation, and issue comments for lightweight coordination between sessions.
